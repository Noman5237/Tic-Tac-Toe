# Tic-Tac-Toe

A simple tic-tac-toe game built using JavaFX.

![image](https://user-images.githubusercontent.com/31593633/126861167-04016633-59ff-4002-aec5-786967302c49.png)

## Run the project

### Requirements
* Java 11 or latest must be installed


For Mac/Linux

```
./gradlew run
```

For Windows

```
./gradle.bat run
```
